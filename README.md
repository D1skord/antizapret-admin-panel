# Админ-панель AntiZapret

Монолитное веб-приложение для управления VPN-сервером AntiZapret.
Стек: **Go (Golang)** + **Vue 3**.
Особенность: компилируется в **единый бинарный файл** (Linux amd64), содержащий внутри себя скомпилированный фронтенд.

---

## 🚀 Установка (Linux)

Установка и обновление выполняются одной командой.
Скрипт автоматически:
1. Скачает последний релиз.
2. Установит бинарник в `/usr/local/bin/antizapret-admin`.
3. Настроит и запустит systemd-сервис `antizapret-admin.service`.

Выполните в терминале сервера:

```bash
curl -sSL https://raw.githubusercontent.com/d1skord/antizapret-admin-panel/main/install.sh | sudo bash
```

После установки панель будет доступна по адресу: `http://<IP-ВАШЕГО-СЕРВЕРА>:8080`

### Удаление
Чтобы полностью удалить приложение и сервис:
```bash
sudo antizapret-admin-uninstall
```

## 🛠 Для разработчиков

### Требования
*   Go 1.25+
*   Node.js 20+
*   Make

### Первый запуск
Установка зависимостей (Go модули + NPM пакеты):
```bash
make install-deps
```

### Локальная разработка
Запустите в двух разных терминалах:

**Терминал 1 (Backend):**
Запускает Go-сервер с hot-reload (через `air`).
```bash
make dev-backend
```

**Терминал 2 (Frontend):**
Запускает Vite сервер.
```bash
make dev-frontend
```
*   Бэкенд: `http://localhost:8080`
*   Фронтенд: `http://localhost:5173` (проксирует API запросы на 8080)

### Сборка релиза (Вручную)
Если вы хотите собрать "боевой" архив для Linux (так же, как это делает GitHub Actions):
```bash
# 1. Собрать бинарник для Linux
make build-release

# 2. (Опционально) Создать структуру для деплоя вручную, взяв файлы из папки deploy/
```
Бинарный файл `antizapret-admin-linux-amd64` появится в корне.

## 🏗 Архитектура проекта

*   `main.go` — Точка входа. Встраивает `frontend/dist` через `embed`.
*   `frontend/` — Vue 3 приложение (Vite).
*   `internal/` — Логика бэкенда (API, работа с файлами конфигурации VPN).
*   `deploy/` — Файлы для деплоя (systemd unit, uninstall script).
*   `mock_fs/` — Фейковая файловая система для локальной разработки (эмулирует пути `/etc/openvpn` и т.д.).
