Act as a Senior Full Stack Developer (Go + Vue.js). I need to scaffold a monolithic web application for managing a VPN server (AntiZapret wrapper).

**Project Constraints:**
1.  **Environment:** Low-resource VPS (1 CPU, 1GB RAM). No Docker.
2.  **Stack:**
    * **Backend:** Go (Golang) using `gin-gonic/gin`.
    * **Frontend:** Vue 3 (Composition API) + Vite + Tailwind CSS.
    * **Deployment:** Single binary (frontend embedded via `embed`).
3.  **Structure:** Monorepo. Root contains Go code, `/frontend` folder contains Vue code.

**Task:**
Create the essential file structure and code for an MVP "VPN Admin Panel" with Authentication.

**Required Functionality:**
1.  **Backend (`main.go`, `internal/api/handlers.go`, `internal/middleware/auth.go`):**
    * **Auth:** Implement a simple Middleware that checks for an `X-Auth-Token` header.
    * **Config:** Expect an `ADMIN_PASSWORD` environment variable (default to "admin123" for dev).
    * **Endpoints:**
        * `POST /api/login`: Accepts `{password}`, returns `{token}` if correct.
        * `GET /api/check-auth`: Validates token.
        * `GET /api/clients`: (Protected) Return mock VPN clients.
        * `POST /api/clients`: (Protected) Simulate creating user.
        * `DELETE /api/clients/:id`: (Protected) Simulate deleting user.
    * Use `os/exec` placeholders for future AntiZapret bash commands.
    * Serve static files from `frontend/dist` on root `/`.

2.  **Frontend (`frontend/`):**
    * Configure `vite.config.js` with proxy.
    * Setup Tailwind CSS.
    * **State Management:** Use a simple Composable (or Pinia if preferred) to store the `isAuthenticated` state and Token.
    * **Axios Interceptor:** Automatically attach `X-Auth-Token` to requests. Redirect to `/login` on 401 error.
    * **Components:**
        * `Layout.vue`: Sidebar + Header (only visible if logged in).
        * `Login.vue`: Simple centered card with password input.
        * `Clients.vue`: Table with "Add" and "Delete" actions.
    * **Router:** Protect routes (redirect to `/login` if no token).

**Output:**
Provide the full code for:
1.  `go.mod`.
2.  `main.go` (setup router, apply middleware).
3.  `internal/middleware/auth.go` (the protection logic).
4.  `internal/api/handlers.go` (login logic + client CRUD).
5.  `frontend/src/router/index.js` (navigation guards).
6.  `frontend/src/stores/auth.js` (simple state).
7.  `frontend/src/views/Login.vue`.
8.  `frontend/src/views/Clients.vue`.
9.  `frontend/src/App.vue`.

Focus on clean, production-ready structure for a small app.
